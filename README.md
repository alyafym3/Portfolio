This is a React + Vite single-page portfolio scaffold with light/dark theme, responsive layout, and sample sections.

Quick start (PowerShell):

```powershell
npm install
npm run dev
```

The app will be available at `http://localhost:5173` by default.

Notes:
- Replace placeholder links and content in `src/data/*.js`.
- Add your CV to `public/dummy-cv.pdf` or replace it.
- Add your profile image as `public/profile.png` (recommended 420x420). The Hero section loads `/profile.png`.
